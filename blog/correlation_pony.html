Measures of central tendency in Pony

When looking into the geometry and shape of your data you often want to know information about the middle or center of the data. There are many measures of centrality each with different pros and cons. We'll talk about how to calculate them, and when you may want to use them.

Pony is an exciting language that fulfils an interesting niche.

It is a type safe, object oriented, actor based language. It compiles to native code and gives a whole host of guarantees when writing concurrent code that make it very attractive.

The syntax looks quite readable and the code compiles and runs fairly quickly.

actor Main
new create(env: Env) =>
    env.out.print("Hello, World!")

This snippet defines an Actor, which is kind of like an asynchronous class where Behaviors are asynchronous methods.

use "collections"
use "format"

actor Main
    fun disp(v: I32, fmt: FormatInt = FormatDefault): String =>
        Format.int[I32](v where width = 1, align = AlignRight, fmt = fmt)
    new create(env: Env) =>
        for i in Range[I32](0, 10) do
        env.out.print(disp(i))
        
        end
        env.out.print(disp(1+1))
